在线订餐系统1.0安装说明

-------------------------
安装方法：
服务器需要开启重写功能

把网站源码上传到网站根目录（暂不支持二级目录），在浏览器运行
：http://www.你的域名.com/install自动进入安装程序

后台地址：http://www.你的域名.com/admin.php

默认用户：admin888 admin888
